## Проведите аудит мер по обеспечению безопасности данных

Рассматриваю задачу через призму
- Федеральный закон № 152-ФЗ «О персональных данных»
- PII - Personally Identifiable Information
- PHI — Protected Health Information
- Privacy By Design

Общие проблемы
- Все данные на одном физическом сервере
- Данные хранятся на общем диске
- Отсутствует резервное копирование
- Данные хранятся в открытом виде
- Доступ к данным без разделения доступа. Для хранения PII и PHI используется общее пространство сетевого диска
- Отсутствует защита от массового просмотра
- Отсутствует контроль изменений данных
- Отсутствует ограничения доступа к полям данных (все видят все)
- Есть вопрос про хранения данных на компьютерах сотрудников и тем как они пользуются электронной почтой
- У сотрудников ресешпена слишком широкий доступ они имеют доступ к PII и PHI данным. При этом все видят все 
- Нужна процедура при которой клиент может удалить свои данные
- Нужна процедура при которой клиент может получить все свои данные

Персональные данные пациента
- Вопрос к расширенной форме сбора данных, нужны ли эти сведения для предоставления услуги
- Запрашивается ли согласие на обработку PII
- Отсутствует политика обработки PII (хранение, удаление)

Запись пациента к специалисту
- В журнале не должно быть данных PII

Данные пациента (анализы, обследования)
- Сотрудник ресепшана не должен иметь доступ к медицинским данным: карте пациента, результатам анализов и исследований
- Политика отношения данных PHI строже. Нужно применять меры защиты от неправомерного доступа, изменения, копирования, распространения и уничтожения данных
- Данные должны шифроваться при передаче и хранении
- К данным должна быть применена псевдонимезация

Прием и обработка платежей
- Проблем с ограничением доступа не вижу
- Есть вопрос с тем как требуется хранить фин.данные и связь персональных данных с фин.данными

Учёт материальных средств
- Наименее проблемная область

Обмен электронной почтой
- Нет контроля пересылаемых сообщений

Список подходящих программных средств
- Symantec DLP или McAfee DLP - решения для предотвращения утечки данных (DLP) сканирование систем на предмет содержания данных по шаблону
- Data Discovery Platforms (BigID или Informatica) - поиск конфиденциальных данных
- Informatica Data Catalog - автоматизация тегирования на основе правил
- BigID и Informatica Axon - поиск нетегированных данных
- OneTrust - контроль согласиями пользователей
- Keycloak - контроль доступа к данным
